About the Project

The aim of the Project is to visualize Topic Modelling on Twitter Data using LDA Algorithm . Here, GNIP API will be used to obtain a huge data-set of tweets from Twitter and stored in Datastax Cassandra. The tweets will further be processed in Apache Spark with three nodes where they are tokenized. Tokens shall be processed using LDA algorithm on Apache Spark. The results will be stored back in Cassandra.

To visualize the obtained results, a User Interface is developed using HTML, CSS, JQuery and D3. The User Interface shall retrieve data from Cassandra through Django Framework.
